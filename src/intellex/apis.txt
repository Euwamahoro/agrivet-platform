title: Rwanda Administrative Locations API | Districts

description: Description
List of Administrative Districts in Rwanda. You have to pass "Provincecode" in the headers with the actual province code (for example: "05"). Results contain a bunch of metadata, but the most relevant are "code", "name", and "description". The "code" will be later used to retrieve subsequent sectors in a separate API endpoint. This API is maintained by Irembo, the Rwandan government's services portal, keeping an accurate and latest database of administrative locations.

URL
GEThttps://api.intellex.dev/traffic/pt/eb89b84d-f562-4410-b336-60a8f6754e5c

Query Parameters
No query parameters
Headers
Provincecode : Example: "05" (Pass in the code for the province you are interested in.)
